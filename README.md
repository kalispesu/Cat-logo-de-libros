Catálogo de Libros

Un sistema de catálogo de libros desarrollado en Java que consume la API Gutendex para buscar libros del dominio público y permite guardarlos en una base de datos local.

Características

- Búsqueda en Gutendex API - Acceso a 60,000+ libros gratuitos
- Base de datos SQLite - Almacenamiento local de libros favoritos
- 5+ opciones de interacción - Menú completo con funcionalidades
- Estadísticas - Análisis de tu biblioteca personal
- Búsqueda local - Filtrado por título, autor o género
- Interfaz de consola - Fácil de usar

Requisitos

- Java JDK 11+
- GSON 2.10.1 (incluido en lib/)
- SQLite JDBC 3.40.0.0 (descargar separadamente)
